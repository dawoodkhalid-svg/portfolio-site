My Projects

Here are some projects I've worked on during my studies in Networking and IT Security. Each project represents skills I've developed and challenges I've overcome.

Network Security Lab Environment

Built a virtualized network security lab using VirtualBox and pfSense firewall. Configured VLANs, implemented firewall rules, and set up IDS/IPS monitoring to simulate real-world enterprise network security scenarios.

Technologies: pfSense, VirtualBox, Snort IDS, Wireshark

Penetration Testing Report

Conducted authorized penetration testing on a virtual vulnerable machine (VulnHub). Performed reconnaissance, vulnerability scanning, exploitation, and privilege escalation. Documented findings and provided remediation recommendations.

Technologies: Kali Linux, Nmap, Metasploit, Burp Suite

Secure Web Application

Developed a secure login system implementing password hashing, SQL injection prevention, and XSS protection. Applied OWASP Top 10 security principles to ensure the application follows security best practices.

Technologies: HTML5, CSS3, PHP, MySQL, Security Headers

Network Traffic Analysis Tool

Created a Python script to analyze network traffic captures and identify suspicious patterns. The tool detects port scans, unusual protocols, and potential data exfiltration attempts from PCAP files.

Technologies: Python, Scapy, Wireshark, Regular Expressions

IT Security Awareness Website

Designed and built a responsive educational website teaching basic cybersecurity concepts to non-technical users. Covered topics like phishing, password security, and social engineering with interactive examples.

Technologies: HTML5, CSS3, JavaScript, Responsive Design
